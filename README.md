## 📖 О проекте

Интернет-магазин разработан для компании **ООО «Модные детки»** — российского поставщика детской одежды, осуществляющего продажи на зарубежный рынок (Беларусь, Украина, Индия, Япония, США, Великобритания и другие страны).

До внедрения данного решения компания вела работу через разрозненные инструменты: каталог в Excel, переписку по Gmail и банковские переводы. Разработанная система объединяет весь процесс — от просмотра каталога до оплаты — в единую платформу.

---

## 🎯 Цель проекта

Разработка полнофункционального интернет-магазина с нуля с использованием HTML, CSS, JavaScript и PHP без применения CMS или конструкторов сайтов, что обеспечивает максимальную гибкость, скорость работы и удобство SEO-продвижения.

## 🗄️ База данных

Схема БД включает 7 таблиц:

| Таблица | Описание |
|---|---|
| `products` | Товары (название, цена, категория, бренд, описание, изображение) |
| `categories` | Категории товаров |
| `brands` | Бренды |
| `cart` | Корзина (привязка к IP) |
| `customers` | Покупатели (имя, email, страна, адрес и др.) |
| `orders` | Заказы (товар, покупатель, количество, статус) |
| `payments` | Оплаты (сумма, транзакция, дата) |
| `admins` | Администраторы |

## 👥 Роли пользователей

### 🧑 Гость
- Просмотр каталога, категорий и брендов
- Добавление товаров в корзину
- Регистрация / авторизация

### 👤 Зарегистрированный пользователь
- Всё что может гость, плюс:
- Оформление и оплата заказов через PayPal
- Просмотр своих заказов и их статусов
- Редактирование персональных данных и пароля
- Удаление аккаунта

### 🔐 Администратор
- Управление товарами (добавление, редактирование, удаление)
- Управление категориями и брендами
- Просмотр и удаление пользователей
- Просмотр всех заказов и изменение их статуса
- Просмотр платёжных транзакций

## 🚀 Установка и запуск

### Требования
- XAMPP (Apache + MySQL + PHP)

### Шаги

1. Клонировать репозиторий в директорию XAMPP:
   ```bash
   git clone https://github.com/DenisFlow/Ecommerce.git C:/xampp/htdocs/ecommerce
   ```

2. Запустить XAMPP и включить сервисы **Apache** и **MySQL**.

3. Открыть [http://localhost/phpmyadmin](http://localhost/phpmyadmin) и создать базу данных с именем `ecommerce`.

4. Импортировать SQL-схему (создать таблицы: `products`, `categories`, `brands`, `cart`, `customers`, `orders`, `payments`, `admins`).

5. Открыть магазин в браузере:
   ```
   http://localhost/ecommerce/
   ```

6. Панель администратора:
   ```
   http://localhost/ecommerce/admin_area/
   ```

## 💳 Платёжная система

Проект интегрирован с **PayPal Sandbox** для тестирования платежей. После успешной оплаты:
- покупателю отправляется письмо с деталями заказа;
- заказ сохраняется в БД со статусом `In Progress`;
- администратор может изменить статус на `Completed`.
